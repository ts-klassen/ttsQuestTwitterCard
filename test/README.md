# テスト

##　ローカル環境でのテスト環境の構築
ローカル環境で動作を確認するには、HTMLファイルの一部を書き換える必要があります。

`replace.sed`は
```
s,置換前,置換後,g
```
の形式で記述されています。ただし、
`\\`はエスケープ記号のため`\\\\`、
`,`は区切り記号のため`\\,`、
`&`は置換前を意味するため`\\&`
と置き換える必要があります。


sedコマンドを使える場合、このディレクトリ内で次のコマンドを走らせることでHTMLファイルを書き換えることができます。
```
sed -f replace.sed ../page.html > page.html
sed -f replace.sed ../audio.html > audio.html
```
